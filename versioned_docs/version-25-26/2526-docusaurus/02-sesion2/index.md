Bienvenida a la segunda sesión del curso sobre Docusaurus. En esta etapa elevamos el nivel técnico para centrarnos en flujos de trabajo avanzados, potenciar la creación de contenidos y automatizar procesos, alineándonos con la filosofía *Docs-as-Code*.

## Objetivos de la sesión

Durante esta sesión abordaremos las siguientes competencias técnicas:

- **Profundización en MDX**: Integración de componentes y *props* dentro de los archivos Markdown para enriquecer la experiencia.
- **Diagramas Visuales**: Uso de formato Mermaid para renderizar gráficos y flujos dinámicamente (*Diagrams as Code*).
- **Generación de Contenido mediante IA**: Aprovechamiento de Antigravity para orquestar la arquitectura documental y automatizar tareas repetitivas.

:::tip Consejo de sesión
Mantén tu servidor de desarrollo local ejecutándose (`npm run start`). Los cambios estructurales en MDX o los diagramas se previsualizarán al instante y facilitarán la validación visual de los apuntes.
:::
