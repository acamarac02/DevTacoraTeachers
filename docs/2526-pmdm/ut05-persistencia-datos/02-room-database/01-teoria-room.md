---
title: "Teoría: Bases de datos con Room"
sidebar_position: 1
description: "Introducción a bases de datos relacionales en Android y el uso de Room como capa de abstracción sobre SQLite."
---

En este documento se presentarán las bases de la persistencia de datos robusta en local con **Room** (una capa de abstracción oficial de Google sobre SQLite). Se repasarán conceptos como qué es una Entity (Tabla), un DAO (Data Access Object) y la clase contenedora Database. Se enseñará a los alumnos el diseño y modelado de datos relacionables dentro de un entorno móvil utilizando anotaciones.
