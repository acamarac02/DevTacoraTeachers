En esta actividad final de la sesión, comprobaremos que tu "túnel" de comunicación entre tu ordenador y GitHub funciona a la perfección. Realizarás cambios locales y los subirás a la nube siguiendo el flujo profesional.

## Objetivo de la Actividad

Consolidar el flujo de trabajo diario de Git (`Modificar` -> `Add` -> `Commit` -> `Push`) asegurando que los cambios se reflejan correctamente en el repositorio remoto.

## Instrucciones Paso a Paso

1.  **Modifica** uno o varios ficheros que ya tengas en tu carpeta de docs.
2.  **Crea** uno o varios ficheros nuevos en la carpeta de docs.
3.  **Sincroniza** el repositorio de GitHub: Ejecuta el flujo completo en la terminal:
    ```bash
    git add .
    git commit -m "Finalizada Sesión 2"
    git push
    ```
4.  **Verifica**: Entra en tu repo de GitHub y comprueba que los archivos y el mensaje del commit aparecen correctamente.
