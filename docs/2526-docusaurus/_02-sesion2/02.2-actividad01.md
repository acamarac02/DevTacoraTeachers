En esta actividad, aplicarás todo lo aprendido en la primera parte de la Sesión 2 para transformar un documento de texto plano en una unidad didáctica profesional y visualmente atractiva utilizando Docusaurus.

## Objetivo de la Actividad

Construir una nueva página de apuntes (puedes inventar el tema, por ejemplo: "Fundamentos de Base de Datos" o "Instalación de Servidores") que integre elementos avanzados para mejorar la experiencia del alumno.

## Requisitos de la Tarea

Debes crear un nuevo archivo `.md` o `.mdx` en tu proyecto (puedes llamarlo `actividad-practica.md`) y asegurarte de que incluya los siguientes elementos:

### 1. Organización con Componentes MDX
Utiliza el componente de **Pestañas (Tabs)** para mostrar información que pueda agruparse de forma lógica. 
*   *Ejemplo*: Si es de programación, muestra el código en diferentes lenguajes. Si es de sistemas, muestra los pasos para Windows y para Linux en pestañas separadas.

### 2. Bloques de Código Profesional
Inserta un bloque de código que utilice metadatos para mejorar su presentación.
*   *Requisito*: El bloque debe tener un **título** descriptivo y **líneas resaltadas** para señalar la parte más importante del código.

### 3. Notación Científica (KaTeX)
Incluye al menos una **ecuación matemática o fórmula** relevante para el tema elegido. 
*   *Idea*: Una fórmula de una ley física, una operación aritmética o la sintaxis de una función. No olvides usar la sintaxis `$...$` para fórmulas en línea o `$$...$$` para bloques destacados. Puedes pedirle la fórmula a ChatGPT o al agente.

### 4. Diagramas por Código (Mermaid)
Representa un proceso o jerarquía mediante un **diagrama Mermaid**.
*   *Requisito*: Debe ser un diagrama de flujo (`graph TD` o `LR`) que explique los pasos de una configuración o el funcionamiento de un concepto teórico.
*   *Truco*: Puedes pedirle a ChatGPT o al agente que te genere el código Mermaid, no hace falta que lo piques a mano.

### 5. Diagramas Externos (Whimsical)
Crea un diagrama utilizando el plugin de [Whimsical](https://whimsical.com/) para ChatGPT e insértalo en tu página como Iframe.

### 6. Integración de Recursos Externos
Inserta un **Iframe** que aporte valor al contenido. Puede ser:
*   Un vídeo de YouTube con un tutorial complementario.
*   Un mapa interactivo de Google Maps.
*   Un cuestionario de Google Forms.

:::tip[¡Dale creatividad!]
No te limites a copiar los ejemplos de los tutoriales. Intenta aplicar estos recursos a una unidad que realmente vayas a impartir en clase para que sea más realista.
:::
