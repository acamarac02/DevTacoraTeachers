# Configuración de Navegación

En el paso anterior aprendimos a crear las carpetas de nuestros módulos (PIA y PMDM). Sin embargo, si entras a tu web ahora mismo, ¡verás que no hay forma de navegar hacia ellos! 

De nada nos sirve tener los apuntes bien estructurados si nuestros alumnos no pueden encontrarlos. En Docusaurus, existen dos piezas clave que rigen cómo nos movemos por la web: la **Barra Superior (Navbar)** y los **Menús Laterales (Sidebars)**. Vamos a enlazarlas.

## Paso 1: La Barra Superior (`docusaurus.config.js`)

El menú principal de Docusaurus es la barra superior. Es el lugar perfecto para poner botones que lleven directamente a cada uno de nuestros módulos.

Abre tu archivo raíz **`docusaurus.config.js`**, busca la sección `themeConfig` y dentro de ella, `navbar`. En la propiedad `items`, configuraremos nuestros dos botones principales:

```javascript
navbar: {
  title: 'Mi Web Docente',
  logo: {
    alt: 'Logo',
    src: 'img/logo.png',
  },
  items: [
    { 
      type: 'docSidebar', 
      sidebarId: 'piaSidebar', // Identificador clave para el Paso 2
      position: 'left', 
      label: 'Módulo de PIA' 
    },
    { 
      type: 'docSidebar', 
      sidebarId: 'pmdmSidebar', // Identificador clave para el Paso 2
      position: 'left', 
      label: 'Módulo de PMDM' 
    },
    // Puedes dejar otros enlaces como el de GitHub a la derecha
    { href: 'https://github.com/...', label: 'GitHub', position: 'right' },
  ],
},
```

La propiedad mágica aquí es **`sidebarId`**. Le estamos diciendo a Docusaurus: *«Cuando el alumno pinche en el botón "Módulo de PMDM", ábrele el submenú lateral llamado "pmdmSidebar"»*. 

Pero... un momento, ¿qué es `pmdmSidebar`? Todavía no existe. Para crearlo, tenemos que pasar al Paso 2.

---

## Paso 2: Menús Laterales Independientes (`sidebars.js`)

Por defecto, Docusaurus mete TODOS los archivos de tu carpeta `docs/` en un único e inmenso menú lateral. Si enseñas tres módulos distintos, tus alumnos verían el temario revuelto. Queremos que el alumno de PIA vea solo temas de PIA.

Abre el archivo **`sidebars.js`** en la raíz de tu proyecto. Vamos a transformarlo para definir *varios* menús laterales, uno por cada módulo. El nombre del menú debe coincidir exactamente con el `sidebarId` que pusimos en el paso anterior.

Sustituye todo el contenido del archivo por esto:

```javascript
// @ts-check

/** @type {import('@docusaurus/plugin-content-docs').SidebarsConfig} */
const sidebars = {
  // 1. Menú lateral para PIA
  piaSidebar: [
    {
      type: 'autogenerated',
      dirName: '2526-pia', // Carpeta exacta dentro de docs/
    },
  ],

  // 2. Menú lateral para PMDM
  pmdmSidebar: [
    {
      type: 'autogenerated',
      dirName: '2526-pmdm', // Carpeta exacta dentro de docs/
    },
  ],
};

export default sidebars;
```

**¿Qué hace este archivo?**
Es un "conector". El botón de la barra superior llama a `piaSidebar`. Y `piaSidebar` dice: *"Docusaurus, autogenera un índice leyendo todo lo que haya dentro de la carpeta `2526-pia`"*.

A partir de este instante, esos dos mundos están conectados.
