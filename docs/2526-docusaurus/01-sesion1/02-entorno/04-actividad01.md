Una vez que conoces la estructura del "cerebro" de tu web (`docusaurus.config.js`) y cómo se gestiona el estilo, ¡es hora de practicar!

## Personaliza tu página

Sigue estos pasos para darle tu toque personal al curso:

1.  **Metadatos**: Cambia el **título** y el **tagline** (subtítulo) de tu sitio.
2.  **Landing Page**: Personaliza la página de inicio. Modifica los textos y las imágenes que aparecen por defecto para que hablen de tu materia.
3.  **Footer**: Modifica el pie de página para incluir tu nombre o enlaces de contacto.
4.  **Colores**: Cambia el color principal de tu web editando las variables CSS en el archivo `custom.css`.

:::info[Aviso sobre Navegación]
De momento no te preocupes por la barra de navegación (navbar), ya que la configuraremos en profundidad más adelante.
:::

:::tip[¡Usa el Live Reload!]
Mantén el comando `npm run start` ejecutándose mientras haces estos cambios. Verás cómo la web se actualiza automáticamente en cuanto guardas los archivos.
:::
