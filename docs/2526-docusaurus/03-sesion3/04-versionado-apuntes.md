title: "Versionado de Apuntes"
sidebar_position: 5
description: "Estrategia para retener el histórico de apuntes de diferentes cursos académicos."
---

Docusaurus nos permite versionar la documentación con un solo comando. Mostraremos cómo aplicar las estrategias de versionado semántico para un entorno educativo: esto implica "congelar" los temarios del año académico en vigor (ej: *Curso 24/25*) y crear simultáneamente el borrador maestro (*next*) del curso siguiente, blindando el acceso web al histórico para el alumnado repetidor.
