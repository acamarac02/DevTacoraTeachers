En este apartado documentaremos el proceso completo para automatizar la publicación de nuestros apuntes. En lugar de compilar el sitio localmente, configuraremos un flujo de trabajo (Workflow) con GitHub Actions que construirá y desplegará automáticamente la documentación de Docusaurus en GitHub Pages de forma pública con cada nuevo *commit*.
